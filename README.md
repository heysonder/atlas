# Atlas

A native iOS YouTube client built on [Piped](https://github.com/TeamPiped/Piped).
SwiftUI + Liquid Glass, targeting iOS 26.

## Features

- **Feed** ‚Äî subscriptions, an on-device "For You" mode, watched filtering, and
  configurable Shorts handling.
- **Channels and search** ‚Äî channel pages, subscriptions, query suggestions, and
  video/channel search.
- **Player** ‚Äî fullscreen or embedded AVPlayer with PiP, AirPlay, background
  audio, SponsorBlock, resume, a session queue, and tappable comment timestamps.
- **Info sheet** ‚Äî channel subscribe, feedback, playlist saving, description,
  comments, and the current queue.
- **Playlists** ‚Äî local playlists from card menus, the Info sheet, and App
  Shortcuts.
- **Downloads** ‚Äî offline video and caption downloads.
- **Profile** ‚Äî history, playlists, downloads, Topic Cloud, backup/restore,
  SponsorBlock settings, instance setup, and privacy policy.
- **App Shortcuts** ‚Äî search, resume, open feed/downloads, play, download, find
  videos, and add to playlist.

## Architecture

- **`PipedKit/`** ‚Äî Codable models, `PipedClient`, instance directory, and
  stream-selection logic.
- **`Atlas/`** ‚Äî the SwiftUI app, organized as:
  - `App/` ‚Äî entry point, `@Observable` `AppModel`, and root view
  - `Models/` ‚Äî SwiftData `@Model` types (one per file: subscriptions, playlists, history)
  - `Features/` ‚Äî feature-grouped screens (Feed, Channels, Search, Player, Downloads, Playlists, Profile, Recommendations)
  - `Components/` ‚Äî reusable views (`VideoRow`, `Thumbnail`, `Avatar`, `CreatorChannelControl`, `GroupedVideoList`, ‚Ä¶)
  - `Support/` ‚Äî non-UI helpers (`Format`, `LoadPhase`, `ThumbnailPrefetcher`, `YouTubeCollaborators`)
- **`project.yml`** ‚Äî source of truth for the Xcode project (generated by [xcodegen](https://github.com/yonaskolb/XcodeGen)).

## Build & run

```sh
# 1. (re)generate the Xcode project
xcodegen generate

# 2. open it
open Atlas.xcodeproj

# 3. select your iPhone, set the signing team if needed, ‚åòR
```

Or from the CLI for the simulator:

```sh
xcodebuild -project Atlas.xcodeproj -scheme Atlas \
  -destination 'generic/platform=iOS Simulator' build CODE_SIGNING_ALLOWED=NO
```

Signing team and bundle id live in `project.yml` (`DEVELOPMENT_TEAM`, `PRODUCT_BUNDLE_IDENTIFIER`).

## Configuration

Atlas does not ship with a default Piped instance. Set one at runtime in
**Profile ‚Üí Settings ‚Üí Instance** before using online video features.
